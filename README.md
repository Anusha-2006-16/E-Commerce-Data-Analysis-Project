📌 Project Title

E-Commerce Sales & Delivery Analysis

🎯 Objective

This project analyzes retail e-commerce data to understand sales trends, delivery performance, customer behavior, and return patterns in order to improve business decision-making.

🛠 Tools Used
Excel (Data cleaning & transformation)
SQL (Data analysis & querying)
Power BI (Dashboard & visualization)
📊 Key Analysis Performed
Sales trend analysis (monthly & yearly)
Delivery delay analysis by shipping mode
Customer segmentation (age, gender, region)
Return rate analysis
Product category performance
🔍 Key Insights
Standard shipping has highest delay rate
Maharashtra contributes highest revenue
Senior customers generate most sales
January shows peak sales
Dairy category has lowest performance
North & East regions are top contributors
📈 Power BI Dashboard
KPI overview (Sales, Profit, Orders, Rating)
Sales trend analysis
Delivery performance dashboard
Customer & regional insights
📌 Outcome

This project helped identify key business inefficiencies in delivery and product performance and provided actionable insights for improving revenue and customer satisfaction.
